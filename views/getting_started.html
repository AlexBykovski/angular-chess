chessboard.js / chess.js directives for AngularJS

Getting started in a minute

As easy as one line

HTML
<nywton-chessgame
  data-game="game" 
  data-board="boardA"
></nywton-chessgame>

Customize everything the angular way

Set all chessboard.js config attributes in your html

HTML
<nywton-chessboard
  data-board="boardB" 
  data-spare-pieces="true"
  data-position="'start'"
></nywton-chessboard>
